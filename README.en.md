## 功能和特点
- 快速浏览网页
>复制链接进入软件即可快速进入

- 能够自定义网页设置
>可关可开，不会保存，UI简洁

- 快速搜索
>自带各种搜索引擎，如百度、bing等

- UI简洁
>给您舒适的浏览体验

- 广告杀手
>屏蔽谷歌广告、浮动广告等阻碍浏览的广告
